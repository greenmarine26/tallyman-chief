# 그린마린 수석검수 통계앱 (PC용)

평택항 양하/선적 통합 실시간 통계

## 주요 기능
- 모든 검수원의 작업 실시간 통계 ☁
- 검수원별 처리량/근무시간
- 규격별/특수화물/POD별/X-RAY 분석
- Firebase 실시간 자동 업데이트

## 설치
```bash
npm install
npm run dev
npm run build
```

- PC 브라우저에서 항상 켜둠
- 자동으로 모든 검수원 작업 표시
- 별도 입력 없음 (읽기 전용 + 항차 삭제만)
